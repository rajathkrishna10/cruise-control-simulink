# cruise-control-simulink
#  Cruise Control System using PID Controller

## Overview
This project simulates a cruise control system using MATLAB Simulink. The system maintains a constant vehicle speed using a PID controller.

## Features
- Feedback control system
- PID based speed regulation
- Disturbance handling (simulated slope)
- Stable and smooth response

##  Concepts Used
- Control Systems
- PID Controller
- Transfer Function
- Feedback Mechanism

##  Results
- System reaches desired speed smoothly
- Handles disturbance effectively
- Minimal steady state error

## Tools Used
- MATLAB Simulink

## Future Scope
- Adaptive Cruise Control (ACC)
- Real time implementation
